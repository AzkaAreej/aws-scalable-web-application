# aws-scalable-web-application
Scalable AWS web application using EC2, Auto Scaling, Application Load Balancer, Amazon RDS, Amazon S3, CloudWatch, and VPC networking.

## Overview

This project demonstrates the deployment of a scalable and highly available web application on Amazon Web Services (AWS). The architecture follows AWS best practices by separating networking, compute, storage, and database layers.

## Architecture

- Amazon VPC
- Public & Private Subnets
- Internet Gateway
- NAT Gateway
- EC2
- Amazon RDS (MySQL)
- Application Load Balancer
- Auto Scaling Group
- Amazon S3
- CloudWatch
- IAM

## Features

- Highly available architecture
- Auto Scaling based on CPU utilization
- Load balancing across multiple EC2 instances
- Secure database deployment in private subnets
- Static website hosting using Amazon S3
- Infrastructure monitoring using CloudWatch
